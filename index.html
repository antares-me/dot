dotfiles

Для корректной работы необходимо установить дополнительные пакеты:

  • cowsay
  • ranger
  • awesome
  • tmux
  • w3m
  • vim
  • zsh
  • git

sudo apt-get install cowsay ranger vim vim-nox awesome awesome-extra git zsh zsh-common w3m w3m-img curl tmux lightdm ubuntu-session xterm emacs cloc libncurses5-dev libncursesw5-dev terminology libssl-dev xserver-xephyr
